TowerGame
=========

Description
-----------
A project for WSUV CS 447, Fall 2014. An isometric, networked, co-op puzzle game.
The premise is to start at the bottom of a tower and navigate up the tower by solving puzzles.

Controls
--------
- WASD/Arrow keys - Movement
- E - Use
- Space - Nothing
